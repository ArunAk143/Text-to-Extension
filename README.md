# Text-to-extension
Text to Extension is a project developed during a 2-month internship at Zaalima Development. It converts user-provided text prompts into functional browser extensions, simplifying extension creation and enabling users to turn ideas into working code quickly and efficiently.

# 🚀 Text to Extension
A full-stack project that converts text into a browser extension using AI.

---

## 📌 Features
* 🧠 Convert text prompts into extension code
* ⚡ Fast frontend (React / Vite)
* 🔧 Backend API integration
* 🔐 Secure environment variable handling
* 📦 Downloadable extension output

---

## ⚙️ Setup Instructions
### 🔽 1. Clone or Download

```bash
git clone https://github.com/Shivam05-sys/Text-to-extension.git
```

OR download ZIP from GitHub and extract.

---

### 📦 2. Install Dependencies
#### Frontend:
```bash
cd extensio-frontend
npm install
```

#### Backend:
```bash
cd extensio-backend
npm install
```

---

### 🔐 3. Setup Environment Variables
Create a `.env` file inside **extensio-backend**
```env
OPENAI_API_KEY=your_api_key_here
```

⚠️ Never share your API key publicly.

---

### ▶️ 4. Run the Project
#### Start Backend:
```bash
cd extensio-backend
npm start
```

#### Start Frontend:
```bash
cd extensio-frontend
npm run dev
```

---

## 🌐 Usage
1. Open frontend in browser
2. Enter your prompt
3. Generate extension
4. Download and use

---

## ⚠️ Important Notes
* `.env` file is not included for security reasons
* You must add your own API key
* Make sure backend is running before using frontend
